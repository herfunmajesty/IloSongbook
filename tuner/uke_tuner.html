How to use Choose a string, play the reference tone, tune until the needle is centered, then switch to the next string.
Best results Pluck one string at a time, keep the room quiet, and tune with the string ringing naturally instead of muting it hard.
Offline friendly The tone generator uses only the browser audio engine, so it works even if microphone access is blocked.
